# Kynox UI Library

Fork of [Fluent](https://github.com/dawid-scripts/Fluent) for Kynox scripts.

## Load in Roblox

```lua
local Fluent = loadstring(game:HttpGet("https://raw.githubusercontent.com/kynox-devx/ui-lib/main/main.lua"))()
local SaveManager = loadstring(game:HttpGet("https://raw.githubusercontent.com/kynox-devx/ui-lib/main/SaveManager.lua"))()
local InterfaceManager = loadstring(game:HttpGet("https://raw.githubusercontent.com/kynox-devx/ui-lib/main/InterfaceManager.lua"))()
```

## Files

| File | Description |
|------|-------------|
| `main.lua` | Core UI library |
| `SaveManager.lua` | Config save/load |
| `InterfaceManager.lua` | Theme & interface settings |
